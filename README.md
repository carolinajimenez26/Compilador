# Compilador

Compilador de miniGo

Se compone de 3 módulos:

- Análisis léxico *(golex.py)*
- Análisis sintáctico *(goparser.py)*
- Análisis semántico *(gocheck)*
- Generación de código intermedio *(gocode.py)*
